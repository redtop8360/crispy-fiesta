WORKING UltimateDiscordBot

Run:
1. pip install -r requirements.txt
2. PowerShell:
   $env:TOKEN="YOUR_BOT_TOKEN"
3. python main.py

Important:
- Run main.py from inside this folder.
- Enable Message Content Intent and Server Members Intent in Discord Developer Portal.
- Invite with scopes: bot + applications.commands.
- Give bot permissions: Send Messages, Embed Links, Manage Channels, Manage Messages, Moderate Members, Kick Members, Ban Members.
